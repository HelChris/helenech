I created "A Bug's Tale," a blog showcasing the skills I've learned over the first year of my studies. I started the project locally and then moved it to WordPress, creating a custom WP REST API to fetch data. The blog was built using HTML, CSS modules, and JavaScript to ensure a dynamic and validated site.
